## 🌑 Overview
**UniCore** is a comprehensive, institutional-grade University Management System (UMS) built on the **MERN** stack. Designed with a "Corporate Stealth" aesthetic, it streamlines academic administration, faculty management, and student engagement into a single, high-performance node.

> **"Bridging the gap between academic complexity and administrative clarity."**

---

## ✨ Key Features

### 🏛️ Multi-Tiered Governance
- **Admin Command Center**: Complete control over staff, departments, and institutional settings.
- **Faculty & Dept Heads**: Specialized dashboards for curriculum and personnel oversight.
- **Teacher Hub**: Simplified attendance, grading, and course management.
- **Student & Parent Portals**: Real-time access to progress, fees, and academic records.

### 🎨 Premium Design System
- **Dark Graphite Aesthetic**: High-contrast, easy-on-the-eyes interface (`#0c0c0e`).
- **Glassmorphism**: Modern, transparent UI components with smooth transitions.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile governance.

### 🛡️ Secure & Scalable
- **JWT-based Authentication**: Secure node-level identity verification.
- **Role-Based Access Control (RBAC)**: Fine-grained permissions across all 11 departmental sectors.
- **Real-time Synchronization**: Powered by Socket.io for instant notifications.

## 🛠️ Tech Stack
- **Frontend**: React 18, Vite, TailwindCSS, Framer Motion
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Real-time**: Socket.io
- **Styling**: Vanilla CSS + Tailwind

## 🚀 Quick Start

### 1. One-Click Launch
Double-click the **`Launch_UniCore.bat`** file in the root directory. This will:
- Start the Backend (API)
- Start the Frontend (Vite)
- Open the browser to `http://localhost:5173`

### 2. Manual Setup
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/unicore.git

# Install Backend Dependencies
cd UniCore/backend
npm install

# Install Frontend Dependencies
cd ../frontend
npm install

# Run Development Servers
# Terminal 1 (Backend)
cd backend && npm run dev
# Terminal 2 (Frontend)
cd frontend && npm run dev
```
