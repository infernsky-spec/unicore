# UniCore: Academic Governance Network
**Platform Overview & Feature Documentation**

Welcome to the comprehensive documentation for **UniCore**, a next-generation, high-fidelity academic management system built to handle multi-tenant university infrastructures. UniCore merges enterprise-grade data management with an incredibly premium, physics-driven UI/UX.

---

## 🏢 Core Platform Features

### 1. Multi-Tenant University Selection
- **Description:** A highly visual, dynamic selection interface that allows users to choose their specific institution before logging in.
- **Capabilities:** Supports 67+ uniquely identified universities, technical colleges, and private institutions.
- **Design:** Features transparent, cleanly extracted university logos, search filtering with `easeOutExpo` Apple-style reflow physics, and scroll-reveals.

### 2. Super Admin "Oversight Terminal"
- **Description:** The master control room for system administrators.
- **Capabilities:** View global metrics across all registered universities, track active connections, monitor cross-institutional analytics, and manage global settings.

### 3. Core Admin Dashboard (NexaVision)
- **Description:** The localized control panel for a specific university's administration.
- **Capabilities:** Manage student/teacher registries, course matrix mapping, financial fee structures, and campus-wide broadcasts. 

### 4. Educator (Teacher) Interface
- **Description:** The portal for professors and lecturers.
- **Capabilities:** Mark student attendance, input evaluation grades, broadcast course-specific announcements, and review student performance analytics.

### 5. Student & Parent Portals
- **Description:** The end-user dashboard for academic tracking.
- **Capabilities:** Real-time GPA tracking, attendance compliance warnings, fee balance authorization, digital timetable matrices, and access to teacher intel/posts.

## ✨ Next-Generation UI/UX Engineering
UniCore abandons the "dull" aesthetic of traditional educational software. It utilizes an ultra-premium design system powered by **Tailwind CSS**, **Framer Motion**, and custom-built interaction logic.

### Global Aesthetic System
- **Unified Light/Dark Mode:** A flawless toggle system. Dark mode uses a deep `slate-950` with amber radial gradient glows, while light mode uses `slate-50` with organic geometry.
- **Small & Cute Aesthetic:** High border radii (`rounded-2xl`, `rounded-[24px]`), uppercase micro-typography with extreme tracking, and glassmorphic backdrop blurs.

## 🚀 The "Epic" Feature Suite
To ensure user interactions are a "lovely experience," UniCore integrates over 20+ features inspired by the top Web3 and open-source UI libraries (Aceternity UI, Magic UI). These features are globally injected, meaning they manifest across all dashboards simultaneously.

### 1. 3D TiltCards (Physics Hover)
Every metric card (Total Students, Active Nodes, etc.) features fluid 3D physics. When you hover over them, they physically tilt and track your mouse cursor in 3D space (`rotateX`, `rotateY`).

### 2. Global Command Palette (Cmd+K)
Pressing `Cmd+K` anywhere in the app summons a beautifully blurred Spotlight search interface, allowing instant navigation to settings, registries, or dashboards without using the mouse.

### 3. Contextual AI Orb
A glowing, pulsing 3D energy orb rests in the bottom right corner of the screen. Hovering over it expands an "Aura Sync" tooltip that provides real-time contextual help and system status.

### 4. Animated Backgrounds (3D Matrix & Particle Vortex)
- **Dark Mode:** Features a hyper-advanced `Starfield3D` projection that uses Z-axis translation and linear physics to create a deep space travel effect, replacing the previous meteor shower.
- **Light Mode:** Features an ultra-visible, scrolling geometric pulse grid overlaid with a massive, spinning `ParticleVortex` array.

### 5. Hardware-Accelerated Number Tickers
Dashboard metrics do not just snap to a number. As the page loads, the values dynamically count up with smooth cinematic acceleration.

### 6. Evervault Cryptographic Text
The small subtitles on dashboard cards feature a cryptographic scramble effect. When hovered over, the text scrambles into random symbols and characters before cleanly decoding into the actual word.

### 7. Shiny Light-Sweep Text
All page headers (e.g., "Welcome, Frank", "Active Course Nodes") feature a continuous, shimmering sweep of light passing through the typography.

### 8. Neural Sync Ticker
A scrolling, hacker-style marquee at the top of the header displaying fake real-time connection status (e.g., "Global Node Matrix Online • Latency: 12ms").

### 9. Cursor Glow Tracker
A soft, hidden amber radial glow physically tracks your mouse cursor across the screen, illuminating the dark mode background beneath it.

### 10. Dynamic Time-of-Day Greetings
The top-left header reads the user's system clock and dynamically updates the greeting to *"Morning Cycle"*, *"Midday Protocol"*, or *"Evening Oversight"*.

### 11. Animated Page Transitions
All pages smoothly slide and fade into each other when navigating using `AnimatePresence`, eliminating clunky, sudden page loads.

### 12. Haptic Navigation Buttons
Clicking any sidebar link triggers a satisfying physics-based "bounce" (haptic feedback visual) mimicking premium mobile apps.
